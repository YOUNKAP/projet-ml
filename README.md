# Mini-projet pratique Python + VS Code

## Présentation du projet

Ce repertoire présente la mise en place d'un environement de programmation Python pour l'IA.

---

## Installation

### Créer un environement Python

```bash
python -m venv venv
```

### Activation de l'environement

#### Linux / macOS
```bash
source venv/bin/activate
```

#### Windows
```bash
.venv\Scripts\Activate
```


### Installations de bibliothèques
```bash
pip install -r requirements.txt
```
---

## Liste des bibliothèques utilisées

bibliothèques  |  Role
